# Fryserappen


## What it is:
An JavaScript app for fridge storage. Visualize fridge contents, add/edit/remove items. Simple login and items stored in a mongodb database. 

Have you ever wondered: What's in my fridge? Do you buy things that you already have, but you dont know it, because the fridge is a mess? Maybe it's time to easily get an overview, reduce foodspill and get control of the fridges contents?

## Quick start:
Download project.
Open in chosen ide (i.e Visual studio code).
In the root of the project, create a .env file

.env (example):
    - PORT = 3000
    - MONGO_URI = "mongodb://###.#.#.#:#####/fryserdb"

Commandline:
    - npm i
    - npm start

App runs on localhost:3000, or value specified in .env PORT variable
